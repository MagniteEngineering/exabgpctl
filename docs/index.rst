==============================
ExaBGP Controller Documentation
==============================

exabgpctl is wrapper arround `ExaBGP <https://github.com/Exa-Networks/exabgp>`_ function.

* Enable / Disable maintenance
* View processes and neigbours
* Check neighbours connectivity
* Check processes statuses

The latest stable version `is available on PyPI <https://pypi.python.org/pypi/exabgpctl/>`_.

.. code-block:: console

    pip install -U exabgpctl
